# skogai marketplace

Personal Claude Code plugin marketplace, with Codex compatibility notes and a local Codex smoke-test plugin.

## Install

```bash
claude plugin marketplace add git@github.com:skogai/marketplace.git
```

Or via HTTPS:

```bash
claude plugin marketplace add https://github.com/skogai/marketplace.git
```
