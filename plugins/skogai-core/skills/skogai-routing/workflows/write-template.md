<workflow>

<objective>
create or revise a template endpoint for repeatable output structure.
</objective>

<steps>

1. identify the output shape that should repeat.
2. create a template with placeholders for variable content.
3. keep instructions minimal and embedded near the relevant placeholder.
4. avoid duplicating long conceptual guidance from references.
5. add a route to the template only when agents need to discover it directly.

</steps>

<validation>

- the template can be copied and filled.
- placeholders are obvious.
- the template does not require reading unrelated files to understand its shape.

</validation>

</workflow>
